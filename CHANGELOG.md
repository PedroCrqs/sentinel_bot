# Changelog

e este projeto segue [Semantic Versioning](https://semver.org/lang/pt-BR/).

## Como versionar novos releases

Ao criar uma nova versão, adicione uma entrada acima seguindo o padrão:

```
## [vX.Y.Z] — AAAA-MM-DD

### Adicionado
- Nova funcionalidade X

### Melhorado
- Comportamento Y agora faz Z

### Corrigido
- Bug em W

### Removido
- Feature depreciada V
```
